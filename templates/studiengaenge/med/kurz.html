{% if lang == "de" %} Kurzprofil{% else %} Short profile {% endif %}

{% if lang == "de" %}
  • Zweijährig und (in Mathematik) deutschsprachig.
  • Zwei gleichberechtigte Fächer und Bildungswissenschaft; kann mit allen Schulfächern kombinert werden (auch Musik an einer Musikhochschule oder Bildende Kunst an einer Kunsthochschule).
  • Praxisphase (13 Wochen) im dritten Semester.
  • Masterarbeit in einer der beiden Fachwissenschaften, einer der beiden Fachdidaktiken oder Bildungswissenschaft.
  • Kann auch als Erweiterungsfach mit 90 oder 120 ECTS-Punkten studiert werden.

      {% if lang == "de" %} Mathematik{% else %}Mathematics {% endif %}

      {% if lang == "de" %}

      In Mathematik werden in Fachwissenschaft und Fachdidaktik die für das Lehramt an Gymnasien nötigen Inhalte vermittelt, die über den Zwei-Hauptfächer-Bachelor-Studiengang hinausgehen. Wahlmöglichkeiten erlauben eine adäquate Vorbereitung sowohl fachwissenschaftlicher als auch fachdidaktischer Master-Arbeiten, in Fachwissenschaft in einem Schwerpunktgebiet eigener Wahl.

      {% else %} {% endif %}

      Und danach?

      Im Anschluss an den Master of Education bildet der Vorbereitungsdienst eine vollständige Ausbildung zur Lehrer in bzw. Lehrer an Gymnasien.
